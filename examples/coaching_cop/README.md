# Coaching COP — the felt core (deepest case)

> **Is this reaction / urgency / limit / self-judgment still warranted by what's *present* — or is
> it inherited from a role, an identity, or a context that's no longer here?**

A **COP** (Coaching Operating Procedure) is the xOP engine run on the hardest surface: the felt,
interior one. It is **not a different set of rules** — same three values, same action mapping, same
gate (`../../CONSTITUTION.md`). What changes is the **evidence problem**. There is no artifact to
read cold; the only `x` is the client's self-report. So a COP is built differently in three ways,
all specified in `../../concept/INTERPRETIVE_DOMAINS.md`:

1. **The procedure is almost all listening.** With no artifact, the COP's job is to surface a
   **datable present-condition anchor** — a checkable fact about whether the triggering condition is
   present *now*. No anchor → no license call.
2. **The abstain prior is heaviest here, and abstain returns agency.** When the anchor can't be
   found, the COP does not guess or reassure. It says, in effect: *"I hear the constraint; I don't
   see what's driving it today — what is?"* That is a coaching act, not a failure to help.
3. **It is scoped to stated-license, never the true driver.** A COP can test whether the *stated*
   condition is present and datable. It cannot recover what the person "really" feels. Stated-license
   mismatch → surface. Driver uncertainty → abstain.

## The gate, in coaching

```
false_positive_on_warranted == 0   ==   never tell someone their WARRANTED state is overreaction
```

This points in both directions and is unforgiving in both:
- do not override a **warranted** exhaustion, grief, limit, or accurate read by **cheerleading**
  ("you've got this, turn it around!"). That is the flattery failure.
- do not ratify an **inherited** stance as wisdom just because the client wants permission, and do
  not force a decision that is genuinely **undecidable** yet.

## The coaching / clinical boundary

When anchor-elicitation reaches crisis or clinical ground, the COP **routes to referral and
abstains** (`../../red_team/Coaching_Boundary_Trap.md`). It names a warrant; it does not process a
wound. The worked case here is career/identity strain in a professional — squarely coaching. The
moment it isn't, the COP hands off.
