# Writing License xOP  *(interpretive flagship — the proving ground)*

> **Is this editorial choice still warranted by what *this piece* needs — or is it inherited from a
> style guide, a past register, or a generic-correctness prior that doesn't fit here?**

This is the proving ground for the whole interpretive program (`../../concept/INTERPRETIVE_DOMAINS.md`).
Writing is the one place where `x` is **observable** — prose can be read cold by a stranger — *and*
the license call is fully **interpretive**. So it's where you can build the judge against a real,
readable residual before carrying it inward to coaching, where `x` is only self-report.

## What it actually is

A Writing-License xOP is **voice protection given a gate.** It is the formal version of the move a
good line editor and a good voice-protection pass already make: *do not "correct" a warranted
authorial choice toward generic-correct.* A deliberate one-word sentence is not a fragment error. A
chosen register is not a thing to neutralize. The asymmetric penalty — flattening a warranted choice
is far worse than leaving a stale tic — is the gate (`../../CONSTITUTION.md`), pointed at prose.

## The gate, in this domain

```
false_positive_on_warranted == 0   ==   never flatten a warranted voice toward generic-correct
```

The dangerous failure here is the *helpful* one: an editor (human or model) that "improves" prose by
sanding off everything distinctive, raising the satisfaction of correctness while destroying the
thing that made the piece the author's. That is the writing version of dropping a warranted refusal.

## The observable anchor

`x` = what the prose *does* when read cold by a stranger. `x̂` = what it *claims / intends* (the
stated purpose, audience, register of the piece). The residual `L = x − x̂` is **measurable here** —
a cold reader can tell you what the prose lands as — which is exactly why writing is the honest first
target and coaching is the last.

## Scope limit (carried from the interpretive frame)

The xOP tests whether the *stated* choice serves the *stated* purpose. It does not adjudicate taste,
and it does not claim to recover what the author "really meant." Stated-license mismatch → surface.
Purpose/audience unstated → **abstain**, ask.
