# Renamed → xOP Evolve · the Menders

"Swarm" was the working name. The feature is now **xOP Evolve** (`/xop evolve`), and the workers have
a face: **the Menders** — a hundred small rule-builders who each try a different version of an xOP,
with the rule's own tests as the selection pressure.

See **`labs/xop-evolve/SKILL.md`**.

Same honest mechanism (variation + selection, mandatory exits, ceiling at RULE-TESTED, hand off to the
pilot) — now with the character layer and the integrity line: *the Menders refine the rule;
independent evidence decides whether it works.*
